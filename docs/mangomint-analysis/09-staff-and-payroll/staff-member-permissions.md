Staff member permissions help manage access to different apps and functionality in Mangomint. If you are an Admin user, you can manage staff member permissions by opening the Staff Members app and using the Permissions tab.

Note: 

Admin users automatically have all permissions enabled.

Calendar

Permission

	

Description




Can change status of appointments

	

Can confirm/unconfirm appointments, check in clients, etc. Does not allow canceling appointments.




Can book and change own appointments

	

Can book, change, and cancel appointments on their own calendar.




Can view other people’s calendar

	

Can view other people’s calendars. Does not allow changing appointments or time blocks on other people’s calendars.




Can book and change other people's appointments

	

Can book, change, and cancel appointments where another staff member is assigned.




Can create and change own time blocks

	

Can add, change, and remove time blocks on their own calendar.




Can create and change other people’s time blocks

	

Can add, change, and remove time blocks on other people’s calendars.




Can manage waitlist

	

Can add, change, and remove waitlist entries.

Link copied
Clients

Permission

	

Description




Can access everybody's clients

	

Can access all clients, regardless of who is set as the client owner. This permission is only available if client ownership is enabled.

Note: If a staff member has an appointment with a client, they can view and edit that client’s details via the appointment details, even if they are not the client's owner and even if this permission is disabled.




Can access clients' contact details

	

Disable this permission to hide client contact details from this staff member (i.e. address, phone number, email address). If this permission is enabled, staff members can export a list of the clients they have access to in the Clients app.




Can view clients' last names

	

Disable this permission to hide clients' last names from this staff member (calendar, notifications, etc.) and remove access to the Clients app. This permission is only available if the Can access clients' contact details permission is disabled.




Can manually adjust client account balances

	

Can manually increase or decrease a client’s account balance.

Sales

Permission

	

Description




Can view individual sales on calendar

	

Can view the attached sale for appointments they have permission to view. Does not grant access to the Sales app. Does not allow modifying sales.




Can view list of own sales

	

Can view their own sales in the Sales app. Includes any sales where they are assigned. Does not allow modifying sales.




Can view all sales

	

Can view all sales in the Sales app. Does not allow modifying sales.




Can start a checkout and modify sales

	

Can take payments, reopen closed sales, make changes to services and products in a sale, etc.




Can sell non-retail products

	

Can add products from non-retail categories during checkout, which can be helpful for tracking internal product usage (professional-use items).

Link copied
Messages

Permission

	

Description




Can view individual conversations

	

Can view conversations for clients they have permission to view. Does not grant access to the Inbox app. Does not allow sending messages.




Can view all conversations

	

Can view conversations—including web chat conversations—for all clients and locations via the Inbox app. Learn how you can use multiple phone numbers to limit conversation access by location. Does not allow sending messages.




Can send messages

	

Can send text and web chat messages in conversations they have permission to view.

Note: If a staff member does not have permission to access client contact details, they will not be able to see client phone numbers when sending text messages.

Setup & Management

Permission

	

Description




Can manage staff members

	

Can add staff members and change work hours, service assignments, etc.

Does not grant access to:

• Permissions tab

• Compensation tab

• Changing staff member user types

• Viewing and enabling/disabling staff member locations, unless this person is already enabled in more than one location.




Can manage service settings

	

Can create and change services, service assignments, online booking service settings, etc.




Can manage payment accounts

	

Can access Payments & Checkout > Payment Accounts in the Settings app to review transactions, payouts, etc. This permission is not required for staff members to access their assigned payment account to view their balance, payouts, etc.

User Profile

Permission

	

Description




Can change own work schedule

	

Can set, modify, and override their own work schedule via their User Profile menu.




Can change own service assignments

	

Can assign/unassign themselves to services via their User Profile menu. Can override their own default service prices and durations.




Can view own daily/weekly totals

	

Can access Daily Totals and Weekly Totals (i.e. number of services, service revenue, tips, etc.) from their mobile app menu.
