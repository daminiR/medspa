üì± Twilio A2P 10DLC Registration

Status: ‚ùå NOT REGISTERED - SMS blocked by US carriers
Error Code: 30034 - "Message from an Unregistered Number"
Why Required: Since 2023, US carriers require all business SMS (A2P) to come from registered 10DLC numbers. Without registration, messages are blocked or filtered.
Steps to Complete:
  1. Go to Twilio Console
  2. Navigate to: Messaging ‚Üí Services ‚Üí Compliance ‚Üí US A2P 10DLC
  3. Register your business/brand (requires EIN, business address)
  4. Create a messaging campaign (describe use case: appointment reminders, check-ins)
  5. Wait 1-7 days for carrier approval
  6. Associate your phone number with the approved campaign
Temporary Workaround: Use a Toll-Free number (~$2/month) which doesn't require 10DLC registration, OR use a number already associated with an approved A2P campaign.
Cost: ~$4/month brand registration + $1.50/month per campaign
Documentation: Twilio A2P 10DLC Guide

üîó Twilio Webhook Configuration

Status: ‚ö†Ô∏è Requires public URL for production
Current: localhost:3001 (dev only)
Steps to Complete:
  1. Deploy app to production server (Vercel, Railway, etc.)
  2. Go to Twilio Console ‚Üí Phone Numbers ‚Üí Your Number
  3. Set "A MESSAGE COMES IN" webhook to: https://yourdomain.com/api/sms/webhook
  4. Set HTTP method to POST
For Local Testing: Use ngrok to expose localhost: ngrok http 3001
