# Keyboard Shortcuts Reference

Quick reference guide for the Medical Spa Messaging System keyboard shortcuts.

---

## Command Palette

| Shortcut | Action |
|----------|--------|
| `⌘K` / `Ctrl+K` | Open command palette |
| `Esc` | Close command palette |
| `↑` / `↓` | Navigate commands |
| `Enter` | Execute selected command |
| Type to search | Filter commands |

---

## Message Actions

| Shortcut | Action |
|----------|--------|
| `⌘⇧C` / `Ctrl+Shift+C` | Close conversation |
| `⌘S` / `Ctrl+S` | Snooze conversation |
| `A` | Assign conversation to team member |
| `T` | Add tag to conversation |
| `⌘Enter` / `Ctrl+Enter` | Send message and close conversation |

---

## Navigation

| Shortcut | Action |
|----------|--------|
| `/` | Search conversations |
| `\` | Insert quick reply template |

---

## Tips

1. **Command Palette is Your Friend**: When in doubt, press `⌘K` to see all available commands
2. **Recent Commands**: The command palette shows your most recently used commands at the top
3. **Smart Search**: You can search by command name, description, or keywords
4. **Keyboard First**: All actions can be performed without touching the mouse

## Platform-Specific Notes

### macOS
- `⌘` = Command key
- `⇧` = Shift key
- `↵` = Return/Enter key

### Windows/Linux
- `Ctrl` = Control key
- `Shift` = Shift key
- `Enter` = Enter key

---

## Accessibility

All keyboard shortcuts are designed to be accessible and follow platform conventions:

- **Screen readers**: Command palette is fully navigable with screen readers
- **Focus management**: Proper focus trapping when palette is open
- **High contrast**: Visible focus indicators for keyboard navigation
- **No mouse required**: Complete functionality available via keyboard
