## Animation Sequence

### Opening Animation (300ms total)

```
Frame 1 (0ms):
- Wizard invisible (scale: 0.8, opacity: 0)
- Background normal

Frame 2 (100ms):
- Background dims (opacity: 0.5)
- Wizard starts scaling (scale: 0.9, opacity: 0.3)

Frame 3 (200ms):
- Wizard visible (scale: 0.95, opacity: 0.7)

Frame 4 (300ms):
- Wizard fully visible (scale: 1.0, opacity: 1.0)
- Ready for interaction
```

### Step Transition (300ms total)

```
Frame 1 (0ms):
- Current step fully visible

Frame 2 (150ms):
- Current step fades out (opacity: 0)
- Progress bar animates

Frame 3 (300ms):
- New step fades in (opacity: 1)
- Ready for interaction
```

### Closing Animation (200ms total)

```
Frame 1 (0ms):
- Wizard fully visible

Frame 2 (100ms):
- Wizard scales down (scale: 0.9, opacity: 0.5)

Frame 3 (200ms):
- Wizard invisible (scale: 0.8, opacity: 0)
- Background returns to normal
- Settings page fully visible
```

## Summary

Users experience a clean, professional 3-step wizard that guides them through essential automated message setup in under 30 seconds. The design is:

- **Clear**: Each step focuses on one question
- **Informative**: Shows exactly what will be enabled
- **Flexible**: Multiple exit options (skip, close)
- **Reversible**: Can go back or re-run wizard
- **Beautiful**: Smooth animations and gradient design
- **Accessible**: Keyboard nav and screen reader support
- **Responsive**: Works on all devices

The wizard eliminates decision paralysis and gets new users up and running immediately.
