# Notification Sounds

This directory contains audio files for notification sounds.

## Required Files

- `notification.mp3` - Sound played when a new notification arrives

## Setup

1. Add a notification sound file named `notification.mp3` to this directory
2. Recommended: Use a short (0.5-2 seconds), pleasant sound
3. Keep file size small (< 50KB) for quick loading

## Alternatives

You can use free notification sounds from:
- https://notificationsounds.com/
- https://freesound.org/
- Create your own using audio editing software

## Format Specifications

- Format: MP3
- Duration: 0.5-2 seconds
- Sample Rate: 44.1kHz
- Bit Rate: 128kbps or lower
- Volume: Normalized to -3dB peak

## Testing

After adding the sound file:
1. Enable sound in the notification panel
2. Trigger a test notification
3. Verify the sound plays correctly
4. Adjust volume if needed (default: 0.5 or 50%)
