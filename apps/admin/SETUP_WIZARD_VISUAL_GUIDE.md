## Color Scheme

- **Primary Gradient**: Purple (#9333ea) to Pink (#db2777)
- **Step 1 Icon**: Blue (#2563eb) - Calendars
- **Step 2 Icon**: Green (#16a34a) - Messaging
- **Step 3 Icon**: Pink (#db2777) - Money/Sales
- **Progress Bar**: White on gradient background
- **Success Indicators**: Green (#10b981)
- **Pro Tips**: Blue (#3b82f6) background

## Responsive Design

### Desktop (1024px+)
- Modal: 672px wide (max-w-2xl)
- Full content visible
- Buttons side-by-side

### Tablet (768px - 1023px)
- Modal: Slightly narrower with margins
- Content remains readable
- Buttons might stack on smaller tablets

### Mobile (< 768px)
- Modal: Full width with padding
- Content stacks vertically
- Buttons stack vertically
- Smaller font sizes
- Touch-friendly button sizes (48px min height)

## Accessibility

- **Keyboard Navigation**: Tab through buttons, Enter to activate
- **Screen Readers**: All icons have aria-labels
- **Focus Indicators**: Clear focus rings on interactive elements
- **Color Contrast**: WCAG AA compliant (4.5:1 minimum)
- **Skip Links**: "Skip - I'll configure manually" clearly labeled

## Animation Timeline

1. **Wizard Appears**: Fade in overlay (200ms), scale up modal (300ms)
2. **Step Transition**: Fade out old content (150ms), fade in new (150ms)
3. **Progress Bar**: Smooth width transition (300ms ease-out)
4. **Wizard Closes**: Scale down modal (200ms), fade out overlay (200ms)
