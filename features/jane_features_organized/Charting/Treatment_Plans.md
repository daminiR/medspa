Who can view a Treatment Plan?
Any staff member with access to a patient‚Äôs profile can view that patient‚Äôs Treatment Plans including the author‚Äôs name, plan name, diagnosis code, date the plan was created, and how many appointments and charts are attached. However, practitioners will not be able to view charts in the plan that they don‚Äôt have access to.
Who can add or remove a chart from a Treatment Plan?
Any staff member with permission to view shared charts can move shared entries in and out of Treatment Plans.
What‚Äôs the difference between closing and deleting a Treatment Plan?
Deleting a plan removes it completely from the patient‚Äôs charting area. A plan needs to have all charts removed before being deleted and you‚Äôd only want to delete a plan that was created by accident.
Closing a plan means that the plan is completed and chart entries cannot be added to it. However, the plan is still available to be exported, viewed, and reopened.
Who can delete, close, or edit a Treatment Plan?
Only the author of the treatment can close, edit, or delete it.
Who can Export a Treatment Plan?
Any staff member with access to view charts in their staff permissions can export a treatment plan. If there are charts within the plan that they don‚Äôt have permission to view, those will not be included in the export.
Can more than 1 diagnosis be added to the plan?
Yes, you can add up to three diagnosis codes to a treatment plan. The first code you add will be the primary.
While you can add multiple diagnosis codes, we recommend that you stick with one treatment plan per discipline-specific concern. So for example, if a patient is recovering from a car accident and seeing multiple different practitioners at the clinic for different injuries stemming from that accident, they would have separate treatment plans for each injury. This helps you keep track of the number of treatments and the progress for each issue.
Can someone create their own coding system?
Yes, you can
create custom billing codes
in Jane, which allows the clinic to set up their own ICD-10 labels.
Can I change the diagnosis part way through the plan?
Yes, the author of the Treatment Plan can edit it at any time and change the diagnosis.
Can I re-open a plan?
Yes, closed plans can be re-opened so that more entries may be added.
Can you add a chart entry to a closed plan?
No, plans can be re-opened first so that a chart entry may be added.
Does a chart entry need to be associated with an appointment to be added to a plan?
No, however, the appointment will only be counted in the plan if the appointment is arrived and linked with the chart entry. If there isn‚Äôt an appointment connected to the chart entry, Jane will not include the appointment in the count. This is why you may sometimes see a higher chart count than the number of appointments tallied.
For a refresher on how to attach a chart to an appointment, check out the last section of
this guide
.
Does Jane count every appointment state?
No, Jane is specifically looking at arrived (attended) appointments. Jane is also looking to ensure that multiple chart entries that are attached to the same appointment aren‚Äôt counted as multiple visits on the appointment counter.
How do we organize the charts within the plan?
The chart entries will continue to follow the same order as we are used to. Starred chart entries will appear at the top of the list followed by the remaining ones from most newest to oldest.
Is there a way to move multiple chart entries into a plan at once?
Not at this time, but we‚Äôd like to
hear your feedback
if this is something that would be helpful for you.
Is there a report that can be run to review active patients?
You bet there is! Check out our section above on the
Treatment Plan Report
.
Does the Treatment Plans Report include diagnosis codes?
Yes, it does. You can also filter the report by diagnosis code using the search bar at the top of the report.
Can I filter the Treatment Plans Report?
Yes, you can filter the report by plan status and by practitioner name and it will give you all patients with Treatment Plans that the practitioner has created.

Can I send a treatment plan to a patient before their appointment?
While you won‚Äôt be able to share it like a regular chart entry, you can export the plan and send it via a secure document sharing platform or via an encrypted email.
